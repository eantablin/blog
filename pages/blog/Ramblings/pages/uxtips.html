Tips

Generally you're gonna want to maximize your return customers, you'll be doing that by making their experience as smooth and pleasant as possible. Here's some tips to help you maximize retention.

  1. Assume user is an idiot
    • Make UI as straightforward as possible preventing them from getting lost
  2. Repetition makes the dream work
    • Teach user how to use your software via repetition; stick to a format that user keeps encountering on every page
